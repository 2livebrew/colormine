ColorMine

Open-source Java library that makes working with colors easy!

View project on GitHub Download ColorMine

Compare Colors

Calculate Delta E: how similar colors are, as perceived by the human eye.

Explore Relationships

Calculate complementary colors, or appealing tertiary and quadrary color schemes.

Color Profiles

Count the individual pixels in an image, or supply a palette and we'll tell you how image maps to it.

Open-source

Built for and MIT licensed for the community via GitHub.

Portable

The java code base allows you to write once, and run (almost) anywhere', including personal computers, android, and the cloud.
